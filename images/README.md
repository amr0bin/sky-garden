# Photographs

Drop a photo in here to replace a plant's illustration in the tour. The tour looks for a file named after the plant's `id` and, if it finds one, shows it in that plant's panel automatically; if not, it falls back to the botanical illustration. No code changes needed.

- **Naming:** `<id>.jpg` (also accepted: `.jpeg`, `.png`, `.webp`). Example: `hawthorn.jpg`, `st-johns-wort.jpg`, `wild-garlic.jpg`.
- **Shape:** the panel image area is landscape (roughly 3:2). A wide crop looks best; the image is centred and cropped to fill.
- **Rights:** use your own photographs, or images you have a clear licence to use — this keeps the public repository clean of licensing problems.

The plant `id` values are exactly the keys in [`../data/plants.json`](../data/plants.json). To list them quickly, they are: crab-apple-dolgo, blackthorn, hawthorn, elder, rowan, foxglove, heather, yarrow, calendula, chamomile, primrose, comfrey, st-johns-wort, water-mint, wild-garlic, meadowsweet, vervain, valerian, nettle, lemon-balm.
